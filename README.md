## Project setup
```
yarn install
```

### Start dev server
```
yarn dev
```

### Build for production
```
yarn build
```

### Locally preview production build
```
yarn serve
```

### Run your unit tests
```
yarn test:unit
```

### Run your end-to-end tests
```
yarn test:e2e
```

### Lints files
```
yarn lint
```
